Sistema de Infraestrutura Corporativa

Modernização completa da infraestrutura de TI na Automotion com migração de 150+ usuários, implementação de Active Directory e virtualização avançada.

Projeto Concluído 6 meses 150+ usuários impactados

Visão Geral do Projeto

Este projeto envolveu a completa modernização da infraestrutura de TI da Automotion, substituindo sistemas legados por uma arquitetura robusta e escalável baseada em tecnologias Microsoft.

A migração foi planejada e executada em fases, garantindo zero downtime nas operações críticas da empresa. O resultado foi uma redução significativa de incidentes e melhoria substancial na produtividade dos usuários.

Resultado Principal: 40% de redução no tempo de resolução de incidentes e 100% de melhoria na confiabilidade dos sistemas.

Resultados

Usuários Migrados 150+
Redução Incidentes 40%
Downtime 0 horas

Tecnologias

Windows Server 2019 Active Directory Hyper-V PowerShell Group Policy Backup Solutions

Arquitetura Implementada

Servidores Principais

  • Domain Controller Principal (Windows Server 2019)
  • Domain Controller Secundário (Backup)
  • File Server com DFS
  • Backup Server com Veeam

Segurança

  • Políticas de Senha Avançadas
  • Controle de Acesso por Grupos
  • Auditoria de Eventos
  • Backup Automatizado

Virtualização

  • Hyper-V Host Cluster
  • 12 VMs de Produção
  • Live Migration Configurado
  • Storage Compartilhado

Monitoramento

  • SCOM 2019
  • Performance Counters
  • Alertas Automatizados
  • Dashboards Executivos

Fases de Implementação

Fase 1: Planejamento e Análise

Levantamento completo da infraestrutura existente, mapeamento de dependências e criação do plano de migração detalhado.

4 semanas Concluído

Fase 2: Preparação da Infraestrutura

Instalação e configuração dos servidores físicos, setup da virtualização Hyper-V e preparação da rede.

6 semanas Concluído

Fase 3: Implementação do Active Directory

Configuração do domínio, políticas de grupo, estrutura organizacional e integração com sistemas existentes.

8 semanas Concluído

Fase 4: Migração dos Usuários

Migração gradual dos usuários em grupos, treinamento da equipe e suporte durante a transição.

6 semanas Concluído

Fase 5: Otimização e Monitoramento

Implementação de monitoramento, otimização de performance e documentação completa dos processos.

2 semanas Concluído

Principais Desafios

Migração sem Downtime

Necessidade de manter operações críticas funcionando durante toda a migração.

Sistemas Legados

Integração com aplicações antigas que dependiam da infraestrutura anterior.

Treinamento de Usuários

Adaptação da equipe às novas políticas e procedimentos de segurança.

Soluções Implementadas

Implementação em grupos pequenos com rollback rápido em caso de problemas.

Ambiente Híbrido

Mantivemos coexistência temporária entre sistemas antigos e novos.

Programa de Capacitação

Treinamentos práticos e documentação detalhada para todos os usuários.
